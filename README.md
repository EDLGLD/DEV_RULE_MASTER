# DEV_RULE_MASTER
# README

## 開発言語
---------------------------------------------------------------------
- Ruby 3.3.0  
- Rails 6.1.7.6  

## 実行手順
---------------------------------------------------------------------
$ git clone git@github.com:EDLGLD/DEV_RULE_MASTER.git
$ cd DEV_RULE_MASTER
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
